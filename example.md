# TextEdit Markdown Preview

Welcome to the **TextEdit** markdown editor!

## Features

- **Live Preview**: See your markdown rendered in real-time
- **Theme Support**: Automatically adapts to light/dark mode
- **GitHub-Style**: Beautiful rendering similar to GitHub
